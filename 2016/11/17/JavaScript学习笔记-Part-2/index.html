5.函数

KeyWord 1:arguments

arguments永远指向当前函数的调用者传入的所有参数,类似一个Array，但不是Array

对于arguments，我的理解是：用户对于我们定义的函数有可能会输入超出预想的参数，此时需要arguments

KeyWord 2::Rest

rest写在函数参数的最后，前面用…来修饰，表示任意个(多余的)参数,(多余的参数以数组形式返回)

KeyWord 3::变量提升

Javascript中，在函数中定义的变量，在定义后，Javascript引擎会扫描一遍函数体，并将所有的变量提升到函数开头

KeyWord 4::let&const

KeyWord 5::this

KeyWord 6::apply()&call()

关于控制this的指向，我们可以使用函数的apply()和call()方法
对于上面的例子,我们可以这样修正
getAge.apply(person,[]);//19,this指向person,参数为空(Array，表示函数本身的参数)
与apply()类似的还有call()方法，不同的是：apply(this变量，[parameter]),call(this变量，parameter) 如：
Math.max.apply(null,[1,2,3,4]);//4
Math.max.call(null,1,2,3,4);//4

KeyWord 7::map

个人理解：map(*function*)传入参数，抽象且方便的数组遍历(循环)。

KeyWord 8::reduce

reduce的效果：[x1, x2, x3, x4].reduce(f) = f(f(f(x1, x2), x3), x4)

KeyWord 9::filter

filter与map类似，filter(*function*)接受一个函数，依次作用于每一个元素，根据函数返回true或false来决定保留或丢弃该元素

KeyWord 10::sort

sort直接对原数组进行更改！
