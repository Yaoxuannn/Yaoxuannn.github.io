7.面向对象编程与继承

不同于Java和C++，JavaScript中不区分Class(然而ES6有了Class继承…)和Instance，而是使用Prototype(原型)来实现面向对象。

//E.g 1:
var arr = [1,2,3];
//原型链是:arr--->Array.prototype--->Object.prototype--->null
//E.g 2: 函数也是一个对象，So：
function foo(){
    return 0;
}
//原型链是:foo--->Function.prototype--->Object.prototype--->null

此时我们就认为xiaohong和xiaoming继承自Student

一个小问题：
xiaohong.hello() === xiaoming.hello(); //false
更多情况下，我们是希望他们能共享这个函数的。
如果想让对象共享同一个函数 只要把函数移动到这些对象的原型上就行了

这样
xiaohong.hello() === xiaoming.hello(); //返回true
