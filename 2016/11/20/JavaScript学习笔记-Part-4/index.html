    JS与HTML

    8.浏览器对象

    window

    window不仅充当全局作用域(就是我们之前说的那个全局变量),而且表示浏览器窗口。
    window对象有innerWidth，innerHeight属性，可以获取浏览器内部的宽度和高度–是指除去占位元素的净宽高。
    对应的，还有outerWidth和outerHeight属性。
    alert("window inner size: " + window.innerWidth + "x" + window.innerHeight);
    window对象有这些方法：

    alert();//弹出带有一段消息的对话框
    prompt();//弹出可输入的对话框
    confirm();//显示带有一段消息及确认、取消按钮的对话框
    open();//打开窗口
    close();//关闭窗口
    print();//打印当前页面
    focus();//给予一个窗口焦点
    blur();//把键盘焦点从顶层窗口移开

    History

    history记录用户浏览过的页面。并可以实现浏览器前进与后退的功能。

    window.history.back()//返回前一个浏览的页面
相当于window.history.go(-1);
window.history.forward()//前进
相当于window.history.go(1);

    navigator

    navigator对象用来表示浏览器信息，其中最常用的属性有：

    * navigator.appName;  //浏览器名称 
* navigator.appVersion; //浏览器版本
* navigator.language; //浏览器设置的语言
* navigator.platform; //操作系统类型
* navigator.userAgent;  //浏览器设定的User-Agent字符串

    然而，因浏览器版本而编写不同的代码，最好还是不要用navigator.userAgent来获取，而是应该使用这样的：
    var width = window.innerWidth || document.body.clientWidth;
    这样便充分利用了JavaScript对不存在属性返回undefined的特性.

    screen

    screen对象表示屏幕的信息，常用的属性有：

    * screen.width;  //屏幕宽度  (像素)
* screen.height;  //屏幕高度  (像素)
* screen.colorDepth;  //颜色位数 (8,16,24...)

    location

    location对象表示当前页面的URL，如：
    http://www.example.com:8080/path/index.html?a=1&b=2#TOP
    可以用location.href获取整条完整的URL。要获取部分值，location提供了以下属性：

    * location.protocol; //"http"
* location.host;   //'www.example.com'
* location.port;   //'8080'
* location.pathname;   //'/path/index.html'
* location.search;  //'?a=1&b=2'
* location.hash;   //'TOP'

    要加载一个新页面，可以调用location.assign()方法.
    要重新加载当前页面(刷新),调用location.reload()方法。

    document

    document对象表示当前页面。HTML在浏览器中是以DOM形式表示为树形结构，document对象就是整个DOM树的根节点。

    9.操作DOM

    DOM(Document Object Model),是一个树形结构，对DOM节点的操作也就只有：

    更新：更新改DOM节点表示的HTML内容；
遍历：遍历该DOM节点下的子节点，以便进行进一步操作；
添加：在该DOM节点下新增一个子节点，相当于动态增加了一个HTML节点；
删除：将该减点从HTML中删除，相当于删掉该DOM节点的内容以及它包含的所有子节点；

    如要操作一个DOM节点，必须要拿到这个DOM节点，也就是定位，最常用的方法是：
    document.getElementById()和document.getElementsByTagName()以及CSS选择器document.getElementsByClassName().
    以上的方法，只有document.getElementById()返回的是DOM节点(因为ID在一个HTML文档中是唯一的)[object HTMLParagraphElement]，其余的方法返回的总是一组DOM节点[object HTMLCollection]
