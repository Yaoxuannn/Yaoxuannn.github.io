Linux目录处理命令

ls X : 查询目录中的内容
    -a 显示所有文件
    -l 显示详细信息 (快捷方式: ll)
    -d 显示目录属性
    -h 人性化显示文件大小
    -i 显示inode
-----------------------------------------
mkdir  XXX：建立目录 (make directory)
    mkdir -p XXX 递归建立目录
-----------------------------------------
cd XXX : 切换所在目录 (change directory)
    cd      进入根(/root) 目录
    cd ~ 进入家(/home)目录
    cd - 进入上次目录
    cd .. 进入上一级目录
    cd . 进入当前目录    
----------------------------------------------
pwd :  查询所在目录位置 (print working directory)
---------------------------------------------
!rmdir : 删除空目录 (remove empty directories)
!BAD ORDER!
-----------------------------------------------
!ATTENTION! !<rm -rf />! 会使Linux自杀的命令
rm -rf X: 删除文件或目录 (remove)
    -r 删除目录
    -f 强制 (force)
-------------------------------------------
cp X X X:  复制 (copy)
    -r 复制目录
    -p 连带文件属性复制
    -d 若源文件是链接文件，则复制链接属性
    -a 相当于 -p + -d + -r
-------------------------------------------
mv X X : 剪切/重命名 (move)
-------------------------------------------
ln -s X X : 链接 (link)(硬链接)
    -s 创建软链接 (soft)

Linux文件搜索命令

Linux 中的通配符
* 匹配任意内容
？ 匹配任意一个字符
[  ] 匹配任意一个中括号内的字符

Linux 中的逻辑关键字
-a = and 逻辑与
-o = or 逻辑或

--------------------------------------------
locate XXX : 按文件名搜索文件
    配置文件:/etc/updatedb.conf
    数据库:/var/lib/mlocate
    更新数据库: updatedb
---------------------------------------------
whereis XXX : 搜索命令（路径及帮助文档所在位置）
    -b : 只查找可执行文件
    -m : 只查找帮助文件
---------------------------------------------
which XXX : 搜索命令的别名(如果有)及所在路径
---------------------------------------------
find [范围] [搜索条件]
#搜索文件

find / -name
find / -iname 
不区分大小写
find / -user 
按照所有者搜索
find / -nouser 
查找没有所有者的文件(to find junk files)
##例外：内核产生或U盘插入

find / -mtime +10 查找十天前修改的文件
-10    10天内
10    10天当天
+10    10天前
atime 文件访问时间
ctime 文件属性改变时间
mtime 文件内容修改时间

find .(当前目录) -size 25k 查找文件大小为25KB的文件
k 千字节必须小写    M 兆字节必须大写
-25k    小于25KB的文件
25k    等于25KB的文件
+25k    大于25KB的文件
find . -inum 262422 查找i节点为262422的文件

find / -size +20k -a -size -50k
查找目录下，大于20KB并且小于50KB的文件
-a = and  逻辑与
-o = or   逻辑或

find / -size +20k -a -size -50k -exec ls -lh {} \;
查找后执行ls -lh命令
-exec * {} \;
---------------------------------------------
grep [选项] "字符串" 文件名
#在文件中匹配符合条件的字符串,可以使用Regex
-i 忽略大小写
-v 排除指定的字符串

Linux压缩命令

Linux中的压缩格式：
.zip、.gz、.bz2
.tar.gz、.tar.bz2

.zip 格式压缩

zip  压缩文件名 源文件
#压缩文件
zip -r 压缩文件名 源目录
#压缩目录

unzip 压缩文件
#解压缩

！zip可以保存源文件，能压缩目录！
--------------------------------------------
.gz 格式压缩

gzip 源文件
#压缩文件
gzip -r 源目录
#压缩目录下的所有子文件，不能压缩目录

gzip -d 压缩文件
#解压缩文件
gunzip 压缩文件
#解压缩文件

！gz不能保存源文件，不能压缩目录！
-------------------------------------------
.bz2 格式压缩

bzip2 源文件
#压缩文件，不保留源文件
bzip2 -k 源文件
#压缩文件，保留源文件

bzip2 不支持目录操作

bzip2 -d 压缩文件
#解压缩
bunzip2 压缩文件
#解压缩
加-k保留压缩文件
--------------------------------------------

不能压缩目录的.gz和.bz2很鸡肋啊。所以Linux为我们准备了.tar.gz和.tar.bz2这两种格式，他们都先把目录打包成.tar的单一文件包，这样就可以进行压缩啦。

-------------------------------------
打包命令tar

tar -cvf 打包文件名 源文件
    -c 打包
    -v 显示过程
    -f 指定打包后的文件名
    -cvf 

解包命令tar

tar -xvf 打包文件名
    -x 解打包
----------------------------------------
统一起来的.tar.gz压缩与解压缩命令

tar -zcvf (可以指定路径)压缩包名.tar.gz 源文件(空格分隔多文件)
    -z 压缩为.tar.gz格式

tar -zxvf 压缩包名.tar.gz
    -x 解压缩.tar.gz格式
-----------------------------------------
统一起来的.tar.bz2压缩与解压缩命令

tar -jcvf 压缩包名.tar.bz2 源文件
    -j 压缩为.tar.bz2格式

tar -jxvf 压缩包名.tar.bz2
    -j 解压缩.tar.bz2格式
    -z 查看压缩文件(不解压)

通过命令末尾添加 -C 路径 来指定压缩路径(???)
------------------------------------

Linux关机与重启命令

Linux中的系统运行级别
0 关机
1 单用户
2 不完全多用户，(不含NFS服务)
3 完全多用户
4 未分配
5 图形界面
6 重启

使用runlevel查看系统当前运行级别

使用vim /etc/inittab来修改系统默认运行级别

使用date命令来显示当前时间

--------------------------------------
shutdown 命令

shutdown [选项] 时间
    -c：取消前一个关机命令
    -h：关机
    -r：重启
    now 立即执行

其他shutdown命令(Unrecommend)
#  halt
#  poweroff
#  init 0

其他restart命令
#  reboot
#  init 6

其他常用命令

---------------------------------------
1.查询与自动挂载命令

mount
#查询系统中已经挂载的设备

mount -a 
#依据配置文件，自动挂载
#配置文件位置：/etc/fstab
------------------------------------------------------------------
2.挂载命令格式
mount [-t 文件系统] [-o 特殊选项] 设备文件名 挂载点
选项：
    -t 文件系统：加入文件系统类型来指定挂载的类型。如：ext3,ext4,iso9660
    -o 特殊选项：可以指定挂载的额外选项
------------------------------------------------------------------
3.挂载光盘
mkdir /mnt/cdrom/
#建立挂载点

mount (-t iso9660)可省
/dev/sr0 /mnt/cdrom
#挂载光盘
-----------------------------------------------
4.卸载光盘
umount 设备文件名或挂载点
！先退出当前路径！
-----------------------------------------------
5.挂载U盘(U盘被看做硬盘)
fdisk -l
#查看U盘设备文件名

mount -t vfat /dev/sdb1 /mnt/usb/
##Linux默认是不支持NTFS文件系统的
-----------------------------------------------
6.w
看到系统当前时间、运行时间、过去15，30，45的平均负载
以及当前用户信息.
----------------------------------------------
7.who
仅仅看到当前用户的信息.
-----------------------------------------
8.last
#查询当前登录和过去登录的用户信息
#默认读取/var/log/wtmp文件数据
--------------------------------------------
9.lastlog
#查看所有用户最后一次的登陆时间
#默认读取/var/log/lastlog文件数据
--------------------------------------------
