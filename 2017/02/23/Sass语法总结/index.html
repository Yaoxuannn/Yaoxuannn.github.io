1.变量语法

Sass使用$符来声明变量。

对于这些常用变量，我们可以将其抽离到一个变量文件。
接下来，来看一看Sass文件引用的能力。

@import "variables";//可以省略文件后缀名和_

1.声明一个_variables.scss文件来存放我们的变量。
2.接下来在我们要引用_variables.scss的文件里(也被称作宿主文件)，
使用@import指令。

这里有一点，原生的CSS也有@import指令，但他不常用，原因有两点：
1.@import必须放在文件的开头。
2.浏览器在解析 A.css 时，如果 A.css 引用了 B.css ，那么浏览器就会在下载 B.css 完成后再继续解析 A.css ，这个过程浏览器处于一个堵塞的过程，对性能非常不利。

Tips:如果想在Sass中使用原生CSS的@import指令，遵循一下4个既定规则即可：
1.@import后面gender文件名是以.css结尾的时候。
2.@import后面跟 http:// 开头的字符串的时候。
3.@import后面跟的是一个url()函数的时候。
4.@import后面带有media queries的时候。

Sass中的注释

Sass支持双斜线和块注释。

但这两种注释的不同在于，Sass在编译时，会忽略 //Comment… 这种注释，/*Comment…*\/会被保留下来，出现在编译后的CSS文件中。

样式加权

CSS文件版本：

.main {
    font-family: $main-ff;
}
.main .headline {
    font-family: $main-ff;
}

Sass文件版本:

.main {
    font-family: $main-ff;

    .headline {
        font-family: $main-ff;
    }
}

但这种写法对于伪元素不可行：

e.g：

a {
    :hover {
        color: #0f0;
    }
}
/*编译后的CSS*/
a :hover {
    color: #0f0;
}
/*等价于*/
a *:hover {
    color: #0f0;
}

因此，Sass引入父类选择器来解决这个问题。

e.g： 

a {
    &:hover {
        color: #0f0;
    }
}

现在我们想要对之前的两个部分增加对字体大小的限制：

.main {
    font-family: $main-ff;

    .headline {
        font-family: $main-ff;
        font-size: 16px;
    }

    .detail {
        font-size: 12px;
    }
}

我们知道，font-family和font-size都是font的属性,因此我们可使用属性的嵌套，上面的部分代码可以改为：

.headline {
    font: {
        family: $main-ff;
        size: 16px;
    }
}

Sass的变量操作与函数

1.变量操作

如果变量是数值类型的，那么支持对 >= <= < > == != () 的比较和优先级修改，另外运算可以带有单位，这也同时意味着，你不能对不同单位进行运算。

例如：

p {
    height: (320px / 3);
}

2.函数

Sass中的函数分为两种：
一种是其内置的内置函数，即 function
还有则是用户自定义的可重用的代码块，即 mixin，这有点类似C语言中的宏，通过@include来调用.
以组合形式出现的以@extends来调用.

下面我们来定义一个mixin：

@mixin name(params) {
    //Code goes here...
}

以新建一个_mixin.scss文件来集中存储我们的函数为例。

为了使我们的mixin更加灵活，我们可以添加参数。

想象这样的一种情况：

.error {
    color: red;
}
.serious-error {
    color: red;
    border: 1px solid #f00;
}

通过Sass的组合，我们可以这样写：

.error {
    color: red;
}
.serious-error {
    @extends error;
    border: 1px solid #f00;
}

如果error类仅用来继承，我们自然是不希望它输出到CSS文件中，这时只用在其前面加上%即可，在@extends时也要带上%哦.

就像这样：

%error {
    color: red;
}
.serious-error {
    @extends %error;
    border: 1px solid #f00;
}

以上就是Sass的初级语法。
