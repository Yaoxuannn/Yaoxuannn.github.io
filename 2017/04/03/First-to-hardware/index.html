    计算机架构与接口设备[CPU篇]

    计算机的五大单元

    从外观来看，我们的计算机存在输入输出设备以及中央处理器，常见的输入设备有：键盘，鼠标，扫描仪，手写板，触摸屏等等，常见的输出设备有：显示器(屏幕)，打印机，扬声器等等….

    整台计算机最重要的部件当属CPU(中央处理器)了，CPU是一个具有特定功能的芯片，内含微指令集，而CPU内最重要的两个单元就是算术逻辑单元和控制单元.

    显然，我们的CPU的作用就是进行算数和逻辑判断，那么他的数据是从哪里获得的呢？

    这就是内存啦，CPU从内存中读取数据，将数据进行计算、处理之后，在写回内存。

    综合一下，可以用下面的这幅图来代表：

    说说CPU

    CPU目前最常见的分为两种，精简指令集和复杂指令集。

    • 精简指令集

      • 顾名思义，这种类型的CPU内，指令执行时间短，完成的操作很单纯，但性能较佳，典型的为Sun,IBM,ARM等

    • 复杂指令集

      • 这种微指令集与精简指令集相比，每一条指令都可以执行较为复杂和数目众多硬件操作，所以相应的，每条指令执行的时间会较长，常见的为AMD、Intel、VIA等x86架构的CPU

    这里提到了x86，由于上述列出的公司所开发出的x86架构的CPU被大量用于个人计算机，而个人计算机又被称为x86架构计算机。x86—由于最早的Intel开发出的CPU代号为8086，后来有80286，80386等，所以便称这种CPU架构为x86架构。

    不同的x86架构的CPU除了在其整体结构不同以外，在微指令集上也有很多不同，目前新的x86CPU大多含有很多先进的指令集，这些指令集可以减少耗电，加强虚拟化的性能，如：

    • 多媒体：MMX,SSE1-4,AMD-3DNow!
    • 虚拟化：Intel-VT, AMD-SVM
    • 省电：Intel-SpeedStep, AMD-PowerNow!
    • 64/32位兼容技术：AMD-AMD64，Intel-EM64T

    这里我们有提到了32/64位，关于这将在后面补充。

    CPU内进行大量高速的数字逻辑运算，所以它是一个高发热量的组件，在CPU的旁边，总是会有一个风扇来散热。

    所以接下来，我们再来了解一下CPU的性能。

    在CPU微指令集相同的情况下，我们通过频率来比较CPU的性能，简单的说，频率就是CPU每秒钟工作的次数，所以频率越高，CPU在单位时间就可以做更多的事。

    我们常见到的属性：外频和倍频，到底指什么呢？

    所谓外频，指的就是CPU与外部组件进行数据传输/运算的速度，而倍频则是CPU内部用来加速工作性能的倍数，两者相称则为CPU的频率。

    在这里稍微再提一下”超频“，其实就是将CPU的这两个频率通过主板的设定，改的更高一点。但是，由于倍频常在出厂是被锁定，因此一般情况下更改的都是外频。于此带来的问题就是，整个主板的组件的运行频率都会升高，顾更容易造成死机等问题。

    了解到这里，再回过去看一看那张图，我们前面说到CPU的数据来源于内存，而内存和CPU进行数据交换的速率显然就是外部频率了，这样的能力就是总线的功能了。

    现在让我们祭出Intel的芯片架构图( •̀ ω •́ )✧：

    这就是整个主板上面最重要的芯片组了，整个芯片组分为两个桥接器来连接和控制各组件之间的通信。分别是：

    • 北桥：
      • 负责连接速度较快的CPU、内存、显卡等
    • 南桥：
      • 负责连接速度较慢的周边接口，如：硬盘、USB、网卡等。

    继续看总线，我们将北桥成为系统总线，南桥就是所谓输入输出(IO)总线，北桥支持的频率不同，从333-1600MHz都有，这样的频率称为前端总线速度(前端一脸满足)，而每次传输的位数则为总线宽度。

    与总线宽度类似，CPU每次处理的数据量为字组大小，根据CPU的设计分为32位和64位，所以我们所称的32/64位就是依据CPU解析的字组的大小而来的。

    CPU等级支持向下兼容(e.g:686的机器上可以安装386的软件)

    也许当你使用CPU-Z这样的软件测试或者查看自己的CPU时，都会看到这样的参数：Cache

    从图中可以看见，L1-3级的缓存(其中L1分数据缓存和指令缓存) 分别是：4x32KB,4x256KB和6M。

    那么到底什么是缓存，又为什么需要缓存呢？

    之前已经说过，CPU从内存中读取数据，由于CPU的运算速度要比内存快得多，为此就要在传输过程中放置存储器来提高数据传输速度。

    以下参考自：IT之家

    • 一级缓存

      • 集成在CPU内部中，用于CPU在处理数据过程中数据的暂时保存。由于缓存指令和数据与CPU同频工作，L1级高速缓存缓存的容量越大，存储信息越多，可减少CPU与内存之间的数据交换次数，提高CPU的运算效率。但因高速缓冲存储器均由静态RAM组成，结构较复杂，在有限的CPU芯片面积上，L1级高速缓存的容量不可能做得太大。

    • 二级缓存

      • 由于L1级高速缓存容量的限制，为了再次提高CPU的运算速度，在CPU外部放置一高速存储器，即二级缓存。工作主频比较灵活，可与CPU同频，也可不同。CPU在读取数据时，先在L1中寻找，再从L2寻找，然后是内存，最后是外存储器。所以L2对系统的影响也不容忽视。

    • 三级缓存

      • 三级缓存是为读取二级缓存后未命中的数据设计的—种缓存，在拥有三级缓存的CPU中，只有约5%的数据需要从内存中调用，这进一步提高了CPU的效率。其运作原理在于使用较快速的储存装置保留一份从慢速储存装置中所读取数据且进行拷贝，当有需要再从较慢的储存体中读写数据时，缓存(cache)能够使得读写的动作先在快速的装置上完成，如此会使系统的响应较为快速。

    内存概览 - ROM

    结构图已经看过了，我们就继续从实际的华硕的主板看起吧~

    看到没看到没右下角那个长长的条子，就是插内存条的喔~一共有四根呐。

    顺带一提，过去的32位计算机，操作系统最大内存编址为4G，即是说，哪怕你插得再多，你也只能使用4G的内存，而现在的计算机都是64位的，最大的内存编址……是大概17,179,869,184 GB(16000PB) ! 当然，这只是理论值

    我们计算机中的内存主要组件叫做动态随机访问内存(Dynamic Random Access Memory,DRAM),这个家伙只有在通电时才能记录和使用，如果断电，那么很不幸，所有的数据将会消失。

    如果你看得仔细，相信你也一定看到啦，在Intel芯片架构的那张图里，内存条上写着DDR3,这又是什么意思呢？、

    DDR

    DDR,是双倍数据传送速度(Double Data Rate)的缩写，意思是说在一次工作周期中进行两次数据的传递，就有点像CPU的倍频。所以传输频率方面显然要比普通的静态传输要好了。

    目前Intel新一代产品已经支持DDR4内存，(2016.10)
    DDR4内存相比DDR3内存:
    频率更高，最高达4266MH.
    电压更低，1.2/1.35V低电压带来极低的功耗.
    容量更大，单条理论最大可达128GB.

    当你的电脑中插了2根或者4根内存，你就可以开启启用内存双通道(Dual-Channel).

    双通道

    在理解双通道之前，我们还要先稍微了解一下总线的概念，假如说，我们把刚才的那个华硕主板比作是一个城市的话，总线就能像是班车，来回运输数据（Bit）。

    这些线路在同一时间内都仅能负责传输一个Bit。因此，必须同时采用多条线路才能发送更多数据，而总线可同时传输的数据数就称为总线的宽度（width），以比特为单位，总线宽度愈大，传输性能就愈佳。

    我们这样计算总线的带宽（即单位时间内可以传输的总数据数）：总线带宽 = 频率x宽度（Bytes/sec）

    好了，现在再来看一下双通道技术..传统的总线宽度大概是64位，要使得存放在内存中的数据能够读取写入的更快速，因此只有加大总线的宽度，而当芯片组厂家能够将内存汇整到一起时(e.g:一条内存的为64位，则两条内存则是128位)，因此，为了使得效果达到最好，最好使用一模一样内存。

    如果说你看的非常仔细，你就会发现，前方的主板图上的内存插槽的颜色是两两对应的，这样的颜色设计就是为了双通道！

    CPU缓存

    之前我们说过，CPU内有着缓存，这种缓存为了达到与CPU频率速度相当，就要更换为另一种随机存储器，即静态随机访问内存(Static Random Access Memory,SRAM).

    这种内存在设计上使用了较多的晶体管，因此价格较高，且不易做成大容量。但，鉴于其速度快，因此将他集成到CPU的内部，成为一种高速缓存以加快数据访问是一个不错的方式！

    回过头看一看CPU篇的最后，是不是有新的理解了呢？

    RAM

    ROM的全称为只读存储器(Read-Only Memory),这是一种非挥发性的内存，最重要的应用就是BIOS(Basic Input Output System)了，BIOS是一套写在主板上一个内存芯片的程序，这个内存芯片在没有通电时也可以将数据记录下来，也就是ROM了。

    另外，当前手机的固件也是使用的ROM，比如，刷机时经常会看到选择ROM的提示信息。 固件，也就是想软件一样被执行的程序，但他对于硬件内部而言更加重要。

    现在的BIOS是可以升级的，原本使用的都是无法改写的内存，而现在的BIOS都是写在了闪存(Falsh Memory)或EFPROM中的。

    显卡概览

    提到显卡，就要先讲一下PCI-E，即：PCI-Express。是电脑总线PCI中的一种，是针对扩展硬件最重要的插槽之一，由于经常用来扩展图形处理器(也就是我们要说的显卡)，而被俗称为显卡插槽。

    显卡又叫做VGA(Video Graphics Array),最初，显卡主要担任图像显示的‘内存’，因为图像的分辨率和色彩深度等信息要占用内存，所以显卡便有一个内存的容量。

    而现在，由于图像，尤其是3D动画和3D物理游戏使得大量的计算对于原本就已经忙碌的CPU而言是个额外的负担，所以显卡上就多了个用来进行3D加速的芯片，也就是GPU

    PCI-E的速度很快，带宽很高，与CPU直接通信，而显卡又正好是一个需要高速运算的部件，因此显卡的规格就是PCI-Express了，这就是PCI-E的插槽（引脚）。

    需要再补充的是，PCI-E使用的是类似管线的概念来处理的，每条管线都是250MB/s的频宽，也就是说管线越大，速度越快，上图就有从X1-X16的PCI-E规格插槽的引脚。

    一般，X1上也会插一些网卡，USB转接卡，声卡等设备。

    硬盘的物理构成–机械硬盘

    硬盘是最为常见的存储介质,由许多的盘片,机械手臂,磁头,主轴马达所组成,如下:

    所有的数据都写在具有磁性的盘片上,当硬盘通电时,主轴马达会带着磁碟飞速旋转起来,机械手臂便会带着磁头在盘片上进行IO操作.

    要注意的是,机械手臂和盘片的接触是非常细微的空间,若磁头出现污损或者出现抖动,则可能导致数据的丢失或损毁.

    整个盘面就像是被划分成了一个个同心圆,这样便分割出了磁盘的最小存储单位,即扇区(Sector),每个扇区大小为512bytes,这是物理上的限制,不能改变.

    当有多硬盘时,所有盘面上的同一个磁道可以组成一个柱面.

    固态硬盘

    固态硬盘与机械硬盘在存储机制上有很大不同,事实上,称其为固态硬盘仅仅是因为命名习惯问题.

    固态硬盘使用NAND闪存(比如)+适当的控制芯片.

    较为常见的固态闪存就是NAND,小容量的NAND闪存经常被制作成带有USB接口的移动存储设备,也就是人们说的’U盘’

    中央是主控芯片,右侧是5块闪存,左边是SATA数据和电力接口.

    主板

    主板是整个主机最重要的一个部分,在主板上负责通信各个部件的,就是芯片组.

    芯片组一般分为南桥和北桥,北桥负责CPU/RAM/VGA等的连接,南桥负责PCI接口和速度较慢的I/O设备.

    如此多的计算机IO/存储设备,芯片组怎么知道这些设备间的通信呢?

    这个时候就要用到设备的I/O地址,就像是IP地址,芯片组会根据这些’门牌号’来寻找,除此之外,还存在一个IRQ中断机制,IRQ就是各个I/O地址连接到CPU的专门路线,设备会通过IRQ告知CPU自己的运行情况,以方便CPU进行工作分配.

    主板最被人熟知的就是BIOS(Basic Input Output System),BIOS是写在主板上的程序,在开机时运行,并读取CMOS中的数据.

    而CMOS是主板上的一块可读写的并行或串行FLASH芯片，是用来保存BIOS的硬件配置(比如:CPU电压和频率,设备的I/O地址)和用户对某些参数的设定(比如:计算机的时间)。

    主板上有很多设备接口(一般在主机箱的后方),主要有:

    • PS/2 接口 很老了,键盘与鼠标接口,差不多已被USB取代
    • USB接口 速度和带宽可见下表:
    USB版本 带宽 理论速度
    USB 3.1超高速+ 10Gbps 1250MB/s
    USB 3.0超高速 5Gbps 625MB/s
    USB 2.0高速 480Mbps 60MB/s
    • RJ-45 网线接口
    • HDMI接口 支持音频传输的视频传输接口
      • 与DVI、DP接口一起常见于显卡侧板。
