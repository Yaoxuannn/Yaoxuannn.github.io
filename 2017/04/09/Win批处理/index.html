Part 1.简单的批处理语法

  • @
    在批处理文件的开头加上@echo off到底是什么意思呢？
    先来了解一下什么是echo就会明白了

  • echo
    echo 本意就是回复，所以在echo on的状态下，你会在执行batch时看到：

echo的另一用法就是输出内容到stdout，当然你也可以用管道把内容接到文件里呀。

所以你到这里也许会对批处理有新的理解了，就是在命令行里按照_语法规则_从上到下跑一遍嘛。

  • ::
    就是注释语法啦，相当于是C中的/*Comment*/

  • pause
    也就相当于是C++里面的system("pause")啦，（C语言是怎么暂停的…）
    根据系统环境不同会有不同的信息，就比如我的是：

  • :

  • goto
    这两个家伙是一起使用的，:就是要跳转的标签，
    goto后的参数就是标签名称，和所有变成编程语言一样，用于流程控制。当然你也可以直接goto到文件尾来终止脚本

  • %
    用来表示变量

  • if
    如果你经常在terminal工作，你一定对命令的帮助非常熟悉，比方说，执行命令(需要参数)时没加参数，会出现帮助文档，加了help，出现文档，加了/?出现文档.

三行if就可以实现上面的效果了。

if除了进行变量值的判断，还可以在后面加上exist关键字，来判断文件是否存在

    • &

    • <& <
